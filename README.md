## Usage

```
$ git clone git@github.com:ssk9597/Docker_Nuxtjs_TypeScript_CompositionApi_Storybook_Clone.git
$ cd Docker_Nuxtjs_TypeScript_CompositionApi_Storybook_Clone
$ make nuxt
$ make typescript
$ make composition-api
$ make storybook
$ make launch-storybook
```
